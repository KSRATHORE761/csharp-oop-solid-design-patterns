# C# OOP, SOLID Principles & Design Patterns

## 📌 Objective

This repository documents my journey to master Object-Oriented Programming (OOP), SOLID principles, Design Patterns, and Low-Level Design (LLD) using C#.

The goal is not only to understand theoretical concepts but also to practice them by building projects, identifying design problems, and applying better software design techniques.

## 🛠 Learning Approach

For each concept:

1. Learn the theory.
2. Understand the real-world problem.
3. Implement a practical example.
4. Apply validations and good design practices.
5. Refactor the code to improve maintainability.
6. Document the learning using README files.
